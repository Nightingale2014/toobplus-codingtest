Coding Test – Frontend (React)

프론트엔드 코딩 테스트 과제로 제작한 React 기반 웹 페이지입니다.
정적인 페이지 구조를 바탕으로, 모바일 퍼스트(Mobile First) 원칙에 맞춰 레이아웃과 UI 흐름을 설계했습니다.

전체 기능을 과하게 늘리기보다는,
신입 프론트엔드 개발자로서 구조 설계, 컴포넌트 분리, 반응형 대응에 대한 이해를 보여주는 데 집중했습니다.

Tech Stack

React
JavaScript (ES6)
HTML5 / CSS3

Project Overview

프로젝트는 src 디렉토리를 기준으로 컴포넌트와 페이지 단위로 구성했습니다.
Header 및 각 섹션은 재사용 가능한 컴포넌트로 분리하여 관리했고,
이미지 리소스 또한 프로젝트 내부에서 직접 관리해 경로 혼선을 최소화했습니다.

Implemented Features

모바일 환경을 기준으로 한 반응형 레이아웃을 구현했습니다.
모바일 해상도에서 헤더 메뉴 토글이 가능하도록 UI를 구성했습니다.
섹션별 UI를 컴포넌트 단위로 분리해 구조를 명확히 했습니다.
불필요한 로직은 배제하고, 가독성과 유지보수를 고려한 구현을 목표로 했습니다.

Responsive Design

Mobile First 방식으로 레이아웃을 설계했습니다.
태블릿 및 데스크톱 해상도에서도 자연스럽게 보이도록 화면 크기에 따라 레이아웃을 조정했습니다.
CSS Media Query를 활용해 각 해상도에 대응했습니다.

Notes

본 과제는 신입 프론트엔드 코딩 테스트임을 고려해,
모든 기능을 완벽하게 구현하기보다는 구조와 구현 방향성을 보여주는 데 중점을 두었습니다.

각 주요 기능에는 상단 주석을 통해 구현 의도와 간단한 설명을 작성했습니다.

Execution

React build 결과물은 build 폴더에 포함되어 있으며,
index.html 파일을 기준으로 결과물을 확인할 수 있습니다.
